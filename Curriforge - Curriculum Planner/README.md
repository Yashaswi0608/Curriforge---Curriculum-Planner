## üöÄ Quick Start

### 1. Clone & Navigate
```bash
cd "Curriforge - Curriculum Planner"
```

### 2. Set Up Environment Variables
Edit `.env` file and add your Gemini API key:
```
GEMINI_API_KEY=your_gemini_api_key_here
```
> Get your free API key from [Google AI Studio](https://aistudio.google.com/app/apikey)

### 3. Install Dependencies
```bash
cd backend
pip install -r requirements.txt
```

### 4. Run the Server
```bash
python main.py
```

### 5. Open in Browser
Navigate to: **http://localhost:8000**

## üì° API Endpoints

### Authentication
| Method | Endpoint | Description |
|--------|----------|-------------|
| POST | `/api/auth/signup` | Register new user |
| POST | `/api/auth/login` | Login with email/password |
| POST | `/api/auth/google` | Google OAuth login |
| GET | `/api/auth/me` | Get current user profile |
| PUT | `/api/auth/profile` | Update profile |

### Courses
| Method | Endpoint | Description |
|--------|----------|-------------|
| POST | `/api/courses/enroll` | Enroll with AI-generated curriculum |
| GET | `/api/courses/` | Get all courses |
| GET | `/api/courses/dashboard` | Dashboard data |
| GET | `/api/courses/{id}` | Get course details |
| PUT | `/api/courses/{id}/topics/{tid}/toggle` | Toggle topic completion |
| DELETE | `/api/courses/{id}` | Delete course |

### Practice
| Method | Endpoint | Description |
|--------|----------|-------------|
| POST | `/api/practice/generate` | Generate 10 practice questions |
| POST | `/api/practice/submit` | Submit answers for evaluation |
| GET | `/api/practice/history/{course_id}` | Practice history |

### AI Chat
| Method | Endpoint | Description |
|--------|----------|-------------|
| POST | `/api/chat/ask` | Ask AI about courses/learning |
